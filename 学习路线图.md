## 1. 总体学习策略

### 1.1 核心原则

```
┌─────────────────────────────────────┐
│  课本 + 课 + 计划 + 笔记 = 完整闭环  │
└─────────────────────────────────────┘
```

| 角色 | 作用 | 用法 |
|------|------|------|
| **课本** | 系统教材，深度讲解 | 概念理解、看完整示例 |
| **课程** | 实战应用，扩展广度 | 跟着敲、补充课本没讲的 |
| **学习计划** | 行动指南 | 告诉你"先做什么、后做什么" |
| **学习指南** | 知识笔记 | 复习用，查阅用 |

### 1.2 学习节奏（关键！）

| 节奏 | 推荐 | 反例 |
|------|------|------|
| **时长** | 每天 1-2 小时 | 一天 8 小时（第二天就累了） |
| **频率** | 每天或隔天 | 一周突击一次 |
| **跨度** | 4-8 周学完基础 | 一年还在学变量 |
| **练习占比** | 50% 写代码 | 90% 看视频 |
| **复盘** | 每节结束做自测 | 学完就忘 |

### 1.3 黄金学习循环

```
   ┌────── 1. 看学习计划（5 分钟）──┐
   │                                  ↓
   │  5. 完成自测（10 分钟）          │
   │                                  ↑
   ├────── 2. 看视频（30 分钟）──────┤
   │                                  ↓
   │  4. 做练习（30 分钟）            │
   │                                  ↑
   └────── 3. 看书 + 敲代码（40 分钟）┘
```

### 📍 阶段 2：语法进阶（6-8 周）

**目标**：掌握 OOP、文件、并发、正则，能写 500 行项目

| 周 | 学习内容 | P 编号 | 重点 |

| W7-W8 | 面向对象（OOP） | P29-P36 | 类、继承、多态 |
| W9 | 文件操作 | P37-P39 | 读写、JSON |
| W10 | 迭代器、生成器 | P40-P42 | Python 高级特性 |
| W11-W12 | 多线程/多进程 | P43-P50 | 并发编程 |
| W13 | 正则表达式 | P51-P54 | 文本处理 |
| W14 | 标准库 | P55-P62 | os/sys/time/logging |

**OOP 重点**（W7-W8）：

```python
# 必掌握的概念
class Animal:           # 父类
    def __init__(self, name): pass
    def speak(self): pass

class Dog(Animal):      # 继承
    def speak(self):   # 方法重写
        print("Woof!")

# 多态、封装、魔术方法
```

**并发编程**（W11-W12）：

```python
# 多线程 - I/O 密集型
import threading
t = threading.Thread(target=func)
t.start()

# 多进程 - CPU 密集型
import multiprocessing
p = multiprocessing.Process(target=func)
p.start()
```

**产出物**：
- 一个 OOP 项目（如"动物园管理系统"）
- 一个文件批处理工具
- 一个多线程爬虫原型

## 4. 常见学习误区

### ❌ 误区 1：只看视频不写代码

```python
# 错误：看完觉得懂了
看了 P10 视频 → "啊，原来 while 循环是这样的"

# 正确：看完立即动手
看了 P10 视频 → 立即写 5 个 while 循环例子
```

### ❌ 误区 2：陷入"教程地狱"

```
看教程 A → 看教程 B → 看教程 C → ...
        ↘ 一直没动手做项目
```

**解决**：看完一个就做个小项目巩固

### ❌ 误区 3：基础不牢就追新技术

```
学完变量 → 直接学爬虫 → 直接学机器学习
         ↘ 越学越崩溃
```

**解决**：按本路线图走，不要跳

### ❌ 误区 4：死记硬背

```python
# 不要背：str.join(list)  # 这种应该查文档
# 应该背：什么时候用什么数据结构  # 这种是思维
```
