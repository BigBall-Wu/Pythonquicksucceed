## 四、具体学习步骤

### 第 1 步：核心概念（15 分钟）

```python
# 关键认知：Python 变量不是"盒子"装值，是"标签"贴对象

# 实验 1：int 不可变
a = 5
print(id(a))           # 140234567890
b = a                  # b 贴到 5
b = 10                 # 重新贴 10（不是改 5）
print(id(a))           # 140234567890（a 还是 5）
print(id(b))           # 140234567902（不同对象）

# 实验 2：list 可变
a = [1, 2, 3]
b = a                  # b 贴到同一个列表
b.append(4)            # 原地修改
print(a)                # [1, 2, 3, 4] ← a 也变了！
print(id(a))           # 相同 ID
print(id(b))           # 相同 ID
```

### 第 2 步：可变 vs 不可变对象（10 分钟）

| 类型 | 可变？ | 例子 |
|------|--------|------|
| int | ❌ | `x = 5` |
| float | ❌ | `x = 3.14` |
| bool | ❌ | `x = True` |
| str | ❌ | `s = "hello"` |
| tuple | ❌ | `t = (1, 2)` |
| frozenset | ❌ | `fs = frozenset()` |
| **list** | ✅ | `lst = [1, 2]` |
| **dict** | ✅ | `d = {}` |
| **set** | ✅ | `s = {1, 2}` |
| **bytearray** | ✅ | `ba = bytearray()` |

**判断方法**：哈希的可作为 dict key / set 元素的，是不可变。

### 第 3 步：深浅拷贝（10 分钟）

```python
import copy

# 浅拷贝（只复制一层）
original = [[1, 2, 3], [4, 5, 6]]
shallow = original.copy()           # 或 list(original) / original[:]

shallow[0].append(99)              # 修改嵌套对象
print("original:", original)        # [[1, 2, 3, 99], [4, 5, 6]] ← 被改了！
print("shallow:", shallow)          # [[1, 2, 3, 99], [4, 5, 6]]

# 深拷贝（完全独立）
original = [[1, 2, 3], [4, 5, 6]]
deep = copy.deepcopy(original)
deep[0].append(99)
print("original:", original)        # [[1, 2, 3], [4, 5, 6]] ← 不变
print("deep:", deep)                # [[1, 2, 3, 99], [4, 5, 6]]
```

### 第 4 步：函数传参陷阱（10 分钟）

```python
# 默认参数陷阱（mutable default）
def add_item(item, lst=[]):         # ❌ 危险！
    lst.append(item)
    return lst

print(add_item(1))                  # [1]
print(add_item(2))                  # [1, 2]（累积！）

# 修正：用 None
def add_item(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst

print(add_item(1))                  # [1]
print(add_item(2))                  # [2]（独立）

# 传 list 给函数
def process(lst):
    lst.append(99)                  # 原地修改！

my_list = [1, 2, 3]
process(my_list)
print(my_list)                      # [1, 2, 3, 99]（被改！）

# 传副本
process(my_list.copy())
```

### 第 5 步：实战 - 配置管理（10 分钟）

```python
# 场景：函数返回配置字典
def get_config():
    """返回配置（深拷贝避免外部修改）"""
    default = {
        "host": "localhost",
        "port": 8080,
        "options": {"timeout": 30, "retries": 3},
    }
    return copy.deepcopy(default)   # 每次返回独立副本

config1 = get_config()
config1["host"] = "example.com"
config1["options"]["timeout"] = 60

config2 = get_config()              # 不受 config1 修改影响
print(config2["host"])              # localhost
print(config2["options"]["timeout"])  # 30
```

### 第 6 步：常见错误（10 分钟）

```python
# 错误 1：浅拷贝以为深拷贝
a = [[1, 2], [3, 4]]
b = a.copy()
b[0][0] = 99                       # 浅拷贝共享嵌套
print(a)                             # [[99, 2], [3, 4]] ← 被改！

# 错误 2：用 `==` 替代 `is`
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)                        # True（值相等）
print(a is b)                        # False（不同对象）
# is 判定身份（同一对象），== 判定值相等

# 错误 3：默认参数用可变对象
def bad_func(data={}):               # ❌ 共享一个 dict
    data["key"] = "value"
    return data

print(bad_func())                    # {"key": "value"}
print(bad_func())                    # {"key": "value", "key": "value"}！
```

### 练习 1：变量标签实验

```python
a = [1, 2, 3]
b = a
c = a.copy()
a.append(4)
print(f"a: {a}")                    # [1, 2, 3, 4]
print(f"b: {b}")                    # [1, 2, 3, 4]（共享）
print(f"c: {c}")                    # [1, 2, 3]（独立）
print(f"a is b: {a is b}")          # True
print(f"a is c: {a is c}")          # False
```

### 练习 2：深浅拷贝对比

```python
import copy
data = {"users": [{"name": "Alice"}, {"name": "Bob"}]}

shallow = data.copy()
deep = copy.deepcopy(data)

shallow["users"][0]["name"] = "Modified"

print("original:", data)            # 被改
print("shallow:", shallow)          # 被改
print("deep:", deep)                # 不变
```
