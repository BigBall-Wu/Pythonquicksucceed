## 二、为什么先学这一节

这一节不是讲 Python 语法，而是给你一张**全局地图**。
在开始 100 节学习前，看完这节能让你：

- 了解本书和课程的整体结构
- 评估自己的起点和目标
- 制定 8-12 周的学习计划
- 建立项目导向学习的思维

## 三、本节要回答的核心问题

学完这一节，你要能回答这几个问题：

1. **致谢是什么？** 语法/概念定义
2. **怎么用？** 至少 1 个能跑的代码示例
3. **什么时候用、什么时候不用？** 适用场景
4. **常见错误是什么？** 至少 1 个易错点
5. **和上节的关联？** 知识衔接
