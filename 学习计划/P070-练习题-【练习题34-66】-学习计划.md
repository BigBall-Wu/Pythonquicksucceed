### 第 2 步：重点详解

#### 题 4-9：立方推导式
```python
cubes = [value ** 3 for value in range(1, 11)]
for c in cubes:
    print(c)
```

#### 题 4-11：你的比萨（复制）
```python
friend_pizzas = my_pizzas[:]   # 复制（不是引用！）
friend_pizzas.append("pepperoni")
print(my_pizzas)               # 原列表不变
print(friend_pizzas)           # 包含新元素
```

#### 题 5-11：序数
```python
for n in range(1, 10):
    if n == 1:
        suffix = "st"
    elif n == 2:
        suffix = "nd"
    elif n == 3:
        suffix = "rd"
    else:
        suffix = "th"
    print(f"{n}{suffix}")
```

#### 题 6-7：人（嵌套字典）
```python
people = {
    "alice": {
        "first": "Alice",
        "last": "Smith",
        "city": "Beijing"
    },
    "bob": {
        "first": "Bob",
        "last": "Jones",
        "city": "Shanghai"
    }
}

for username, info in people.items():
    print(f"\nUsername: {username}")
    for key, value in info.items():
        print(f"  {key}: {value}")
```
