## 一、函数返回值 return 深度解析

### 1.1 return 的本质

`return` 语句是函数与外部世界通信的核心渠道。它决定了函数执行完毕后能"返回"给调用者的数据。

```python
# 最简单的返回值
def add(a, b):
    return a + b

result = add(3, 5)  # result = 8

# 没有 return 的函数
def say_hello():
    print("Hello!")

result = say_hello()  # result = None
```

### 1.2 return 的三大特性

#### 特性一：终止函数执行

```python
def demo():
    print("第一条")
    return 42  # 函数在这里结束
    print("这条永远不会执行")

print(demo())  # 输出：第一条\n42
```

#### 特性二：返回 None（隐式）

```python
def no_return():
    pass

print(no_return())  # None

# 下列情况都会返回 None
def case1():
    return  # 只有 return，没有值

def case2():
    pass    # 什么都没有

def case3():
    1 + 1   # 有表达式但不返回
```

#### 特性三：返回任意数据类型

```python
def return_int():
    return 42

def return_str():
    return "hello"

def return_list():
    return [1, 2, 3]

def return_dict():
    return {"name": "Alice", "age": 20}

def return_func():
    return lambda x: x * 2

# 函数可以返回函数（高阶函数）
doubler = return_func()
print(doubler(5))  # 10
```

### 1.3 多个返回值的处理

#### 元组打包返回（最常用）

```python
def get_user_info():
    name = "张三"
    age = 25
    return name, age  # 实际返回 (name, age)

# 调用方式一：直接解包
user_name, user_age = get_user_info()
print(f"姓名：{user_name}，年龄：{user_age}")

# 调用方式二：接收为元组
info = get_user_info()
print(info)  # ('张三', 25)
```

#### 字典返回（结构化）

```python
def get_student_record():
    return {
        "name": "李四",
        "math": 92,
        "english": 88,
        "chinese": 95
    }

record = get_student_record()
print(f"{record['name']}的平均分：{(record['math'] + record['english'] + record['chinese']) / 3}")
```

#### 列表返回（有序集合）

```python
def get_all_files():
    return ["config.txt", "data.csv", "report.pdf"]

files = get_all_files()
for f in files:
    print(f"找到文件：{f}")
```

## 二、return 与 print 的本质区别

这是新手最容易混淆的概念！

| 对比项 | return | print |
|--------|--------|-------|
| 目的 | 向调用者传递数据 | 向屏幕显示信息 |
| 返回值 | 有（可被变量接收） | None |
| 函数内部 | 可多次使用，但只有第一个生效 | 可多次使用 |
| 使用场景 | 需要处理或计算结果 | 调试/日志 |

```python
# ❌ 错误示例：混淆 return 和 print
def calculate_sum(a, b):
    print(a + b)  # 屏幕显示，但返回 None

result = calculate_sum(3, 5)
print(result + 10)  # TypeError: None + 10

# ✅ 正确示例
def calculate_sum(a, b):
    return a + b  # 返回值

result = calculate_sum(3, 5)
print(result + 10)  # 18
```

## 三、文档字符串（Docstring）完全指南

### 3.1 为什么需要文档字符串

```python
# 没有文档
def calc_area(r):
        return 3.14 * r ** 2

# 有文档
def calc_area(r):
    """
    计算圆的面积
    
    Args:
        r (float): 圆的半径，单位：米
    
    Returns:
        float: 圆的面积，单位：平方米
    
    Example:
        >>> calc_area(5)
        78.5
    """
    return 3.14 * r ** 2
```

### 3.2 单行 vs 多行文档字符串

#### 单行文档字符串

```python
def abs(n):
    """返回数字的绝对值。"""
    return n if n >= 0 else -n
```

#### 多行文档字符串

```python
def find_max(numbers):
    """
    在给定列表中找出最大值
    
    这是一个更详细的说明，可以写多行。
    主要描述函数的功能和使用场景。
    
    Args:
        numbers: 整数或浮点数的列表
    
    Returns:
        最大值。如果列表为空，返回 None
    
    Raises:
        TypeError: 如果传入的不是可迭代对象
    """
    if not numbers:
        return None
    return max(numbers)
```

## 五、综合实战练习

### 练习一：实现统计器

```python
class Statistics:
    """
    统计数据类
    
    用于计算一组数值的基本统计量
    
    Attributes:
        data: 存储所有添加的数值
    
    Example:
        >>> stats = Statistics()
        >>> stats.add(10)
        >>> stats.add(20)
        >>> stats.add(30)
        >>> print(stats.mean())
        20.0
    """
    
    def __init__(self):
        self.data = []
    
    def add(self, value: float) -> None:
        """添加一个数值"""
        self.data.append(value)
    
    def count(self) -> int:
        """返回数值个数"""
        return len(self.data)
    
    def sum(self) -> float:
        """返回数值总和"""
        return sum(self.data)
    
    def mean(self) -> Optional[float]:
        """返回平均值"""
        if not self.data:
            return None
        return sum(self.data) / len(self.data)
    
    def min(self) -> Optional[float]:
        """返回最小值"""
        return min(self.data) if self.data else None
    
    def max(self) -> Optional[float]:
        """返回最大值"""
        return max(self.data) if self.data else None
    
    def summary(self) -> Dict[str, Optional[float]]:
        """返回所有统计量的字典"""
        return {
            "count": self.count(),
            "sum": self.sum(),
            "mean": self.mean(),
            "min": self.min(),
            "max": self.max()
        }
```

### 练习二：文件处理器

```python
def read_file_lines(filepath: str, encoding: str = "utf-8") -> Dict[str, Union[int, List[str]]]:
    """
    读取文件并返回统计信息
    
    Args:
        filepath: 文件路径
        encoding: 文件编码，默认为 utf-8
    
    Returns:
        包含 'line_count', 'lines', 'total_chars' 的字典
    
    Raises:
        FileNotFoundError: 文件不存在时抛出
    """
    with open(filepath, 'r', encoding=encoding) as f:
        lines = f.readlines()
    
    return {
        "line_count": len(lines),
        "lines": lines,
        "total_chars": sum(len(line) for line in lines)
    }


def write_file_lines(filepath: str, lines: List[str], encoding: str = "utf-8") -> bool:
    """
    写入多行到文件
    
    Args:
        filepath: 文件路径
        lines: 要写入的行列表
        encoding: 文件编码，默认为 utf-8
    
    Returns:
        写入成功返回 True
    """
    with open(filepath, 'w', encoding=encoding) as f:
        f.writelines(line if line.endswith('\n') else line + '\n' for line in lines)
    return True
```

## 七、易错点与避坑指南

1. **忘记 return**：函数计算了结果但没有返回，导致结果为 None
2. **return 和 print 混淆**：print 只显示，不返回
3. **多个 return 只执行第一个**：函数遇到 return 就结束了
4. **文档字符串过时**：修改函数后忘记更新文档
5. **类型提示不是强制**：Python 不会因为类型提示错误而报错
