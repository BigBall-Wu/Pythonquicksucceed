## 四、具体学习步骤

### 第 1 步：常用内置函数（按类别）（15 分钟）

**数值类**：
```python
print(abs(-5))             # 5
print(round(3.14159, 2))   # 3.14
print(pow(2, 10))          # 1024
print(sum([1, 2, 3]))      # 6
print(min(3, 1, 2))        # 1
print(max(3, 1, 2))        # 3
```

**序列类**：
```python
nums = [3, 1, 4, 1, 5, 9, 2, 6]
print(len(nums))           # 8
print(sorted(nums))         # [1, 1, 2, 3, 4, 5, 6, 9]
print(reversed(nums))       # 反向迭代器
print(list(reversed(nums)))  # [6, 2, 9, 5, 1, 4, 1, 3]
print(any([False, True]))   # True
print(all([True, True]))    # True
```

**类型转换**：
```python
print(list("abc"))         # ['a', 'b', 'c']
print(tuple([1, 2]))      # (1, 2)
print(set([1, 1, 2]))     # {1, 2}
print(dict([("a", 1)]))   # {"a": 1}
print(int("42"))           # 42
print(str(42))             # "42"
print(bool(1))             # True
```

**对象相关**：
```python
print(type(42))             # <class 'int'>
print(isinstance(42, int))  # True
print(id(42))               # 内存地址
print(dir([]))              # 列表的所有方法
print(help(str.upper))      # 查看文档
```

### 第 2 步：解包（* / **）的 5 种用法（15 分钟）

```python
# 1. 元组解包
a, b, c = (1, 2, 3)
a, b, c                    # (1, 2, 3)

# 2. 列表解包
x, y, z = [1, 2, 3]

# 3. 用 * 收集剩余
first, *middle, last = [1, 2, 3, 4, 5]
# first=1, middle=[2,3,4], last=5

# 4. 字典解包
d = {"a": 1, "b": 2}
a, b = d.values()           # 解包值
a, b = d                    # 解包键

# 5. 函数调用解包
def add(a, b, c):
    return a + b + c

args = [1, 2, 3]
print(add(*args))           # 6

kwargs = {"a": 1, "b": 2, "c": 3}
print(add(**kwargs))        # 6
```

### 第 3 步：zip 和 enumerate（10 分钟）

```python
# zip：并行遍历
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]
cities = ["Beijing", "Shanghai", "Guangzhou"]

for name, age, city in zip(names, ages, cities):
    print(f"{name} ({age}岁) 在 {city}")

# zip 转字典
keys = ["name", "age", "city"]
values = ["Alice", 25, "Beijing"]
d = dict(zip(keys, values))

# enumerate：同时拿索引和值
fruits = ["apple", "banana", "cherry"]
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")
# 1. apple
# 2. banana
# 3. cherry
```

### 第 4 步：map / filter / sorted + lambda（10 分钟）

```python
nums = [1, 2, 3, 4, 5]

# map：应用函数
squared = list(map(lambda x: x ** 2, nums))
# [1, 4, 9, 16, 25]

# filter：筛选
evens = list(filter(lambda x: x % 2 == 0, nums))
# [2, 4]

# sorted：自定义排序
students = [("Alice", 25), ("Bob", 20), ("Charlie", 30)]
by_age = sorted(students, key=lambda s: s[1])
# [("Bob", 20), ("Alice", 25), ("Charlie", 30)]

# 反向
by_age_desc = sorted(students, key=lambda s: -s[1])
```

### 第 5 步：实战 - 数据处理 pipeline（10 分钟）

```python
# 处理学生数据
students = [
    {"name": "Alice", "score": 85, "age": 20},
    {"name": "Bob", "score": 92, "age": 22},
    {"name": "Charlie", "score": 78, "age": 21},
    {"name": "Dave", "score": 95, "age": 23},
]

# 1. 过滤：及格学生
passed = list(filter(lambda s: s["score"] >= 60, students))

# 2. 映射：提取姓名分数对
pairs = list(map(lambda s: (s["name"], s["score"]), passed))

# 3. 排序：按分数降序
sorted_students = sorted(passed, key=lambda s: -s["score"])

# 4. 取前 3 名
top3 = sorted_students[:3]

for i, s in enumerate(top3, 1):
    print(f"{i}. {s['name']}: {s['score']}")
```

### 第 6 步：常见错误（5 分钟）

```python
# 错误 1：解包数量不匹配
a, b = [1]                    # ❌ ValueError: not enough values
a, b, c = [1, 2, 3, 4]      # ❌ too many values
# 解决：用 * 收集
a, *rest, b = [1, 2, 3, 4]  # a=1, rest=[2,3], b=4

# 错误 2：map 忘了 list
result = map(lambda x: x ** 2, [1, 2, 3])
# result 是 map 对象（迭代器）
print(result)                  # <map object>
# 解决
print(list(result))            # [1, 4, 9]

# 错误 3：sorted 与 sort 混淆
nums = [3, 1, 2]
sorted(nums)                    # 返回新列表 [1, 2, 3]
nums.sort()                     # 原地排序，nums 变 [1, 2, 3]
```
