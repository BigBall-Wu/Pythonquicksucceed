## 七、动手练习

### 练习 1：重现视频示例

把以下示例代码自己手敲一遍：

```python
def count_up_to(n):
    i = 0
    while i < n:
        yield i
        i += 1

# 调用不会执行函数体，而是返回一个生成器
gen = count_up_to(5)
print(type(gen))        # <class 'generator'>

# 每次 next() 执行到下一个 yield
print(next(gen))        # 0
print(next(gen))        # 1
print(list(gen))        # [2, 3, 4]
```
