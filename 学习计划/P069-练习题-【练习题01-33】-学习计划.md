### 第 3 步：重点题目详解

#### 题 2-3：个性化消息
```python
name = "Alice"
print(f"Hello {name}, would you like to learn some Python today?")
```

#### 题 2-6：名言 2
```python
famous_person = "Albert Einstein"
message = '"A person who never made a mistake never tried anything new."'
print(f"{famous_person} once said, {message}")
```

#### 题 3-4 / 3-5：嘉宾名单
```python
guests = ["Alice", "Bob", "Charlie"]
# 3-4
for guest in guests:
    print(f"亲爱的 {guest}，诚邀您参加晚宴。")
# 3-5
print(f"\n{guests[0]} 无法赴约。")
guests[0] = "Dave"
for guest in guests:
    print(f"亲爱的 {guest}，诚邀您参加晚宴。")
```

#### 题 3-6 / 3-7：增删嘉宾
```python
# 3-6
print("找到更大的餐桌！")
guests.insert(0, "Eve")
guests.insert(len(guests)//2, "Frank")
guests.append("Grace")
# 3-7
print(f"无法邀请 {guests.pop()}，抱歉。")
print(f"无法邀请 {guests.pop(0)}，抱歉。")
del guests[0]      # 3-7
del guests[0]
for guest in guests:
    print(f"亲爱的 {guest}，您仍在邀请名单上。")
del guests[0]
del guests[0]
print(guests)        # []
```
