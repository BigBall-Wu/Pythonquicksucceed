## 七、动手练习

### 练习 1：重现视频示例

把以下示例代码自己手敲一遍：

```python
def outer(msg):
    def inner():
        print(msg)          # 引用了外层函数的变量
    return inner

f = outer("hello")
f()                         # "hello"
del outer                   # 即使 outer 被删，inner 仍能访问 msg
f()                         # "hello"
```
