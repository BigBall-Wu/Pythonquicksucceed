## 爬虫分类

| 类型 | 特点 | 例子 |
|------|------|------|
| 通用爬虫 | 抓全网，搜索引擎用 | Googlebot |
| 聚焦爬虫 | 抓特定主题/网站 | 抓取某网站所有商品 |
| 增量爬虫 | 只抓新内容/更新 | 新闻订阅 |
| 深度爬虫 | 跟踪链接，深入抓取 | 学术论文 |

## 按实现技术

| 方式 | 优缺点 |
|------|--------|
| `requests + BeautifulSoup` | 简单静态页面 |
| `requests + lxml/XPath` | 高效解析 |
| `Scrapy` | 大型项目 |
| `Selenium` | JS 渲染页面 |
| `Playwright` | 现代浏览器自动化 |
| `pyppeteer` | 无头 Chrome |

## 选型决策树

```
页面是否需要 JS 渲染？
  ├─ 是 ─ Selenium / Playwright
  └─ 否
      ├─ 项目复杂度？
      │   ├─ 小型 ─ requests + BeautifulSoup
      │   └─ 大型 ─ Scrapy
      └─ 反爬严重程度？
          ├─ 轻微 ─ requests
          └─ 严重 ─ Scrapy + 代理池 + 浏览器指纹
```

```python
# 1. 聚焦爬虫：只爬某个分类下的页面
import requests
from bs4 import BeautifulSoup

def focused_crawler(category_url, max_pages=5):
    """只爬指定分类的页面"""
    results = []
    for page in range(1, max_pages + 1):
        url = f"{category_url}?page={page}"
        resp = requests.get(url, headers={"User-Agent": "Mozilla/5.0"})
        soup = BeautifulSoup(resp.text, "lxml")
        for item in soup.select(".item"):
            results.append({
                "title": item.select_one(".title").text,
                "link": item.select_one("a")["href"],
            })
    return results

# 2. 增量爬虫：只爬新内容
import os
import json
import hashlib

class IncrementalCrawler:
    def __init__(self, state_file="crawled.json"):
        self.state_file = state_file
        self.seen = self._load_state()

    def _load_state(self):
        if os.path.exists(self.state_file):
            with open(self.state_file) as f:
                return json.load(f)
        return {}

    def _save_state(self):
        with open(self.state_file, "w") as f:
            json.dump(self.seen, f)

    def is_new(self, url, content):
        h = hashlib.md5(content.encode()).hexdigest()
        return self.seen.get(url) != h

    def save(self, url, content):
        self.seen[url] = hashlib.md5(content.encode()).hexdigest()
        self._save_state()

crawler = IncrementalCrawler()
resp = requests.get("https://example.com/news")
if crawler.is_new("https://example.com/news", resp.text):
    crawler.save("https://example.com/news", resp.text)
    process_news(resp.text)
```
